# #101 Symmetric Tree

## 递归遍历
判断左右子树是否对称，递归方式遍历  
时间$$O(n)$$  
空间$$O(\log n)$$  
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def isSymmetric(self, root: TreeNode) -> bool:
        if root is None:
            return (True)
        return (self.isSame(root.left, root.right))
    
    def isSame(self, a: TreeNode, b: TreeNode) -> bool:
        if not (a or b):
            return (True)
        if not (a and b):
            return (False)
        if a.val != b.val:
            return (False)
        return (self.isSame(a.left, b.right) and self.isSame(a.right, b.left))
```

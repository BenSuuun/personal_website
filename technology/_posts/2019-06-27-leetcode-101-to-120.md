# #101 Symmetric Tree

## 递归遍历
判断左右子树是否对称，递归方式遍历  
时间$$O(n)$$  
空间$$O(\log n)$$  
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def isSymmetric(self, root: TreeNode) -> bool:
        if root is None:
            return (True)
        return (self.isSame(root.left, root.right))
    
    def isSame(self, a: TreeNode, b: TreeNode) -> bool:
        if not (a or b):
            return (True)
        if not (a and b):
            return (False)
        if a.val != b.val:
            return (False)
        return (self.isSame(a.left, b.right) and self.isSame(a.right, b.left))
```

## Tip
* 注意节点为None的处理

# #102 Binary Tree Level Order Traversal

## 队列层次遍历
遍历当前层次，并将其可行的子节点加入新队列，直到新队列为空  
时间$$O(n)$$  
空间$$O(n)$$  
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def levelOrder(self, root: TreeNode) -> List[List[int]]:
        if root is None:
            return ([])
        output = []
        queue = [root]
        while queue:
            new_queue = []
            val_record = []
            for node in queue:
                val_record.append(node.val)
                if node.left:
                    new_queue.append(node.left)
                if node.right:
                    new_queue.append(node.right)
            output.append(val_record)
            queue = new_queue[:]
        return (output)
```

## Tip
* 需要分割上一层队列与下一层队列

# #103 Binary Tree Zigzag Level Order Traversal

## 队列层次遍历
类似#102遍历当前层次，并将其可行的子节点加入新队列，直到新队列为空。此外，维护一bool判断当前层次是否需要逆序    
时间$$O(n)$$  
空间$$O(n)$$  
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def zigzagLevelOrder(self, root: TreeNode) -> List[List[int]]:
        if root is None:
            return ([])
        queue = [root]
        output = []
        flag = True
        while queue:
            val_list = []
            new_queue = []
            for node in queue:
                val_list.append(node.val)
                if node.left:
                    new_queue.append(node.left)
                if node.right:
                    new_queue.append(node.right)
            queue = new_queue[:]
            if flag:
                output.append(val_list[:])
                flag = False
            else:
                output.append(val_list[::-1])
                flag = True
        return (output)
```

## Tip
* 层次逆序需考虑左子树和右子树本身的逆序

# #104 Maximum Depth of Binary Tree

类似#102遍历当前层次，并将其可行的子节点加入新队列，直到新队列为空。此外，统计遍历的层次数记为深度

空间$$O(n)$$  
```python
# Definition for a binary tree node.

#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def maxDepth(self, root: TreeNode) -> int:
        if root is None:
            return (0)
        queue = [root]
        level = 0
        while queue:
            level += 1
            new_queue = []
            for node in queue:
                if node.left:
                    new_queue.append(node.left)
                if node.right:
                    new_queue.append(node.right)
            queue = new_queue[:]
        return (level)
```

# #105 Construct Binary Tree from Preorder and Inorder Traversal

## 递归构建
前序遍历的第一个点为当前子树的根节点。在中序遍历找到值对应的点后，在中序遍历左侧的元素位于左子树，右侧的元素位于右子树，随后递归构建整棵二叉树。  
时间$$O(n^2)$$  
空间$$O(n\log n)$$   
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def buildTree(self, preorder: List[int], inorder: List[int]) -> TreeNode:
        if not preorder:
            return (None)
        node = TreeNode(preorder[0])
        i = inorder.index(preorder[0])
        node.left = self.buildTree(preorder[1:i+1], inorder[:i])
        node.right = self.buildTree(preorder[i+1:], inorder[i+1:])
        return (node)
```

## Tip
* 列表查找值避免用for语句，index效率能够高一些

# #106 Construct Binary Tree from Inorder and Postorder Traversal

## 递归构建
类似#105的方法，但根节点为后序遍历的最后一个元素。同样在中序遍历中找到该元素的位置，随后左右子树分别遍历构建。  
时间$$O(n^2)$$  
空间$$O(n\log n)$$  
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def buildTree(self, inorder: List[int], postorder: List[int]) -> TreeNode:
        if not postorder:
            return (None)
        node = TreeNode(postorder[-1])
        pos = inorder.index(postorder[-1])
        node.left = self.buildTree(inorder[:pos], postorder[:pos])
        node.right = self.buildTree(inorder[pos+1:], postorder[pos:-1])
        return (node) 
```

## Tip
* 使用index提高检索效率

# #107 Binary Tree Level Order Traversal II

## 队列层次遍历

```python
class Solution:
    def levelOrderBottom(self, root: TreeNode) -> List[List[int]]:
        if root is None:
            return ([])
        queue = [root]
        val_list = []
        while queue:
            new_queue = []
            val_tmp = []
            for node in queue:
                val_tmp.append(node.val)
                if node.left:
                    new_queue.append(node.left)
                if node.right:

```

# #108 Convert Sorted Array to Binary Search Tree

## 递归构建
选取数组的中心作为当前的根节点，其左右分别作为左子树和右子树，递归构建二叉树  
时间$$O(n)$$  
空间$$O(n)$$  
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def sortedArrayToBST(self, nums: List[int]) -> TreeNode:
        if not nums:
            return (None)
        mid = len(nums) // 2
        node = TreeNode(nums[mid])
        node.left = self.sortedArrayToBST(nums[:mid])
        node.right = self.sortedArrayToBST(nums[mid+1:])
        return (node)
```

## Tip
* 注意空节点的处理

# #109 Convert Sorted List to Binary Search Tree

## 预构建结构
由于元素是递增的，相同长度的输入会产生相似的结构性质，与其具体的元素值无关。根据模拟，发现所有层次的左子树都是满二叉树数。因而问题转化为，各层次的左子树元素数目序列是多少。首先遍历整个链表得到元素长度。例如，长度为11的输入，其各层次的左子树加上该层次的根节点数序列应为[4, 4, 2, 1]。这一序列的值可以通过递归计算$$\text{floor}[\log_2 \frac{n}{1/5}]$$的方式求得。在得到这一序列后，维护两个栈，分别表示已经搭建完成所有子树待加入至上层节点的左子树的节点栈，以及左子树已经连接待加入右子树的节点栈。同时加入栈中节点的深度信息，便于递归拼接。针对上述序列，分别得到相应元素数的组块，最后前一组块的右子树设为后一组块，第一个组块的根节点即为整株树的根节点。  
时间$$O(n)$$  
空间$$O(\log n)$$  
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

import math

class Solution:
    def sortedListToBST(self, head: ListNode) -> TreeNode:
        if not head:
            return (None)
        n = 0
        pointer = head
        while pointer:
            n += 1
            pointer = pointer.next
        tmp = n
        num_list = []
        while tmp > 1:
            num_list.append(2 ** (math.floor(math.log2(tmp/1.5))))
            tmp -= num_list[-1]
        pointer = head
        component = []
        for i in range(len(num_list)):
            sub_stack = []
            root_stack = []
            for j in range(num_list[i]):
                if not sub_stack:
                    node = TreeNode(pointer.val)
                    sub_stack.append((node, 0))
                else:
                    node = TreeNode(pointer.val)
                    tmp = sub_stack.pop()
                    node.left = tmp[0]
                    root_stack.append((node, tmp[1]+1))
                while (sub_stack and root_stack and 
                       sub_stack[-1][1] + 1 == root_stack[-1][1]):
                    tmp = root_stack.pop()
                    tmp[0].right = sub_stack.pop()[0]
                    sub_stack.append(tmp)
                pointer = pointer.next
            if num_list[i] != 1:
                component.append(root_stack[-1][0])
            else:
                component.append(sub_stack[-1][0])
        node = TreeNode(pointer.val)
        for i in range(len(component)):
            tmp = component.pop()
            tmp.right = node
            node = tmp
        return (node)
```

# #110 Balanced Binary Tree

## 递归遍历深度
当前节点的深度为左节点深度与右节点深度较大者加1，判断左右子树深度是否相差大于1，并维护一个flag判断是否有任何节点出现不平衡的情况。  
时间$$O(n)$$  
空间$$O(\log n)$$  
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def isBalanced(self, root: TreeNode) -> bool:
        self.balanced = True
        self.depth(root)
        return (self.balanced)
        
    def depth(self, root: TreeNode) -> int:
        if root is None:
            return (0)
        left_depth = self.depth(root.left)
        right_depth = self.depth(root.right)
        if abs(left_depth - right_depth) > 1:
            self.balanced = False
        return (max(left_depth, right_depth)+1)
```

## Tip
* 调试过程输出判断是否正常执行

# #111 Minimum Depth of Binary Tree

## 递归遍历深度
判断当前节点是否为叶节点，若是则返回深度1；否则递归左右子树，返回深度较小的子树加1  
时间$$O(n)$$  
空间$$O(n)$$  
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None
import sys

class Solution:
    def minDepth(self, root: TreeNode) -> int:
        if root is None:
            return (0)
        if not root.left and not root.right:
            return (1)
        left = sys.maxsize
        right = sys.maxsize
        if root.left:
            left = self.minDepth(root.left)
        if root.right:
            right = self.minDepth(root.right)
        return (min(left, right)+1)
```

## Tip
* 注意最小深度的定义，需要合理处理节点为None的情形，包括直接输入为None

# #112 Path Sum

## 递归遍历节点
遍历所有的根至叶子的路径，计算和是否等于所要求值。  
时间$$O(n)$$  
空间$$O(n)$$  
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def hasPathSum(self, root: TreeNode, sum: int) -> bool:
        if root is None:
            return (False)
        if root.left is None and root.right is None:
            if sum == root.val:
                return (True)
            else:
                return (False)
        return (self.hasPathSum(root.left, sum-root.val) or 
                self.hasPathSum(root.right, sum-root.val))
```

## Tip
* 由于未给出元素正负性，不能剪枝

# #113 Path Sum II

## 递归遍历节点
同#112，遍历所有的根至叶子的路径，额外维护一列表存储路径，若满足要求加入输出。  
时间$$O(n)$$  
空间$$O(n)$$  
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def pathSum(self, root: TreeNode, sum: int) -> List[List[int]]:
        self.output = []
        self.pathFind(root, sum, [])
        return (self.output)
        
    def pathFind(self, root: TreeNode, sum: int, path: List[int]):
        if root is None:
            return
        path.append(root.val)
        if root.left is None and root.right is None:
            if sum == root.val:
                self.output.append(path[:])
            else:
                path.pop()
                return
        self.pathFind(root.left, sum-root.val, path)
        self.pathFind(root.right, sum-root.val, path)
        path.pop()
```

## Tip
* 维护路径注意加入与弹出的对应操作

# #114 Flatten Binary Tree to Linked List

## 中序遍历
维护一指针，并通过中序遍历依次将元素链接到指针上。此外，在递归时，需要在内部额外存储节点的左右子树，避免指针操作修改原本的树关系。  
时间$$O(n)$$  
空间$$O(n)$$  
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def flatten(self, root: TreeNode) -> None:
        """
        Do not return anything, modify root in-place instead.
        """
        self.fake_root = TreeNode(0)
        self.node = self.fake_root
        self.inorder(root)
    
    def inorder(self, root: TreeNode):
        if root is None:
            return
        left = root.left
        right = root.right
        self.node.left = None
        self.node.right = root
        self.node = root
        self.inorder(left)
        self.inorder(right)
```

* 注意递归函数中本地变量的使用

# #115 Distinct Subsequences

## 动态规划
记$$f[i, j]$$为S前i个字符和T前j个字符的满足要求的个数，转移方程$$f[i, j] = f[i - 1, j] + f[i - 1, j - 1]\cdot \text{if}\{S[i] = T[j]\}$$。初始值，$$f[0, 0] = 1$$，其余均为0。由于i的递增表现，可以使用滚动数组减小内存开支，但需要逆序更新。  
时间$$O(mn)$$  
空间$$O(n)$$  
```python
class Solution:
    def numDistinct(self, s: str, t: str) -> int:
        f = [0] * (len(t) + 1)
        f[0] = 1
        for i in range(1, len(s) + 1):
            for j in reversed(range(1, len(t) + 1)):
                if i < j:
                    f[j] = 0
                    continue
                if t[j-1] == s[i-1]:
                    f[j] += f[j-1]
        return (f[len(t)])
```

## Tip
* 预先列出转移方程，并模拟判断其正确性
* 使用滚动数组时，注意对应的滚动维度以及其顺序逆序更新要求
* 对于初始化值需要模拟判断

# #116 Populating Next Right Pointers in Each Node

## 宽搜遍历
整体是宽搜的思路遍历，考虑到空间是常数级的，无法维护新的队列，可以利用next的链表关系作为下一层次的访问队列。维护两个相邻层次的指针头与指针即可。  
时间$$O(n)$$  
空间$$O(1)$$  
```python
"""
# Definition for a Node.
class Node:
    def __init__(self, val, left, right, next):
        self.val = val
        self.left = left
        self.right = right
        self.next = next
"""
class Solution:
    def connect(self, root: 'Node') -> 'Node':
        head = root
        while head:
            pointer = head
            lower_head = Node(0, None, None, None)
            lower_pointer = lower_head
            while pointer:
                if pointer.left and pointer.right:
                    lower_pointer.next = pointer.left
                    pointer.left.next = pointer.right
                    lower_pointer = pointer.right
                pointer = pointer.next
            head = lower_head.next
        return (root)
```

## Tip
* 充分利用处理完成的数据得到的信息，减少冗余开支

# #117 Populating Next Right Pointers in Each Node II

## 宽搜遍历
类似#116，宽搜遍历，并利用上一层次的next链表作为访问队列。区别在于需要判断节点是否有左右子树。

```python
"""
# Definition for a Node.
class Node:

        self.left = left

"""
class Solution:

```

# #118 Pascal's Triangle

## 层级遍历
按层次计算，每一层次由上一层次计算得到  
时间$$O(n^2)$$  
空间$$O(n^2)$$  
```python
class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        if numRows == 0:
            return ([])
        output = [[1]]
        for i in range(numRows - 1):
            row = output[-1]
            new_row = [1]
            for j in range(len(row) - 1):
                new_row.append(row[j] + row[j+1])
            new_row.append(1)
            output.append(new_row[:])
        return (output)
```

# #119 Pascal's Triangle II

## 层次遍历
类似#118，按层次计算，并通过滚动数组降低内存开销  
时间$$O(n^2)$$  
空间$$O(n)$$  
```python
class Solution:
    def getRow(self, rowIndex: int) -> List[int]:
        if rowIndex == 0:
            return ([1])
        row = [1]
        for i in range(rowIndex):
            for j in range(i):
                row[j] = row[j] + row[j+1]
            row = [1] + row
        return (row)
```
